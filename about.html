About me

Hai nama saya Muhammad Raditya Alif Nugroho. Saya adalah seorang siswa kelas 10. Hobi saya adalah nyari tau informasi tentang teknologi, programming, videografi, dan video editing. Selain itu, saya juga senang bermain futsal dan gitar. Saya sedang mencoba berbagai hal di dunia teknologi seperti data science, web development, dan mobile development. Website ini adalah proyek pertama saya dan kemungkinan kedepannya akan ada proyek lain.

Saya menyukai sepak bola, tim favorit saya adalah FC Barcelona dan Manchester City. Saat ini saya sedang mempelajari Bahasa Belanda. Saya ingin menguasai lebih banyak bahasa kedepannya, bukan untuk apa-apa, cuma keinginan pribadi.
